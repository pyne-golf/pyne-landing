# Pyne Golf — Landing Page

Landing page and waitlist for [pyne.golf](https://www.pyne.golf). Built with React + Vite + TypeScript.

## Tech stack

- [Vite](https://vitejs.dev/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Supabase](https://supabase.com/) (waitlist storage)

## Local development

Requires Node.js & npm ([install via nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
git clone https://github.com/pyne-golf/pyne-landing.git
cd pyne-landing
npm install
npm run dev
```

## Deployment

Hosted on [Vercel](https://vercel.com). Push to `main` to deploy automatically.

To deploy your own instance, import the repo in Vercel — no configuration needed, it auto-detects Vite.
